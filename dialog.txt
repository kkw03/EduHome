' ══════════════════════════════════════════════════════════════
' UC5: Update Market Data (System Background Process)
' ══════════════════════════════════════════════════════════════


state "UC5 · Update Market Data (System Timer)" as SYS {


 [*] --> IdleState


 IdleState : entry / Initialise timer monitoring
 IdleState : do / Await scheduled trigger from system timer
 IdleState : exit / Begin data retrieval process


 IdleState -right-> RetrievingMarketData : scheduledTimerTriggered


 RetrievingMarketData : entry / Connect to HDB and URA data sources
 RetrievingMarketData : do / Download latest resale transaction data
 RetrievingMarketData : exit / Pass raw data to validation


 RetrievingMarketData -right-> ValidatingMarketData : [data downloaded]
 RetrievingMarketData --> ErrorMarketData : [download failed]


 ValidatingMarketData : entry / Check data format and structural integrity
 ValidatingMarketData : do / Verify no corruption or unexpected schema
 ValidatingMarketData : exit / Pass validated data to database update


 ValidatingMarketData -right-> UpdatingDatabase : [data valid]
 ValidatingMarketData --> ErrorMarketData : [data invalid]


 ErrorMarketData : entry / Log format or validation error details
 ErrorMarketData : do / Abort current update run
 ErrorMarketData : exit / Return to idle and await next schedule


 ErrorMarketData --> IdleState : awaitNextSchedule


 UpdatingDatabase : entry / Write new transactions to internal database
 UpdatingDatabase : do / Refresh derived analytics and PSF values
 UpdatingDatabase : exit / Signal watchlist check to proceed


 UpdatingDatabase -right-> CheckingWatchlists : [new data persisted]
 UpdatingDatabase --> NoNewDataState : [no new data]


 NoNewDataState : entry / Log that database is already up to date
 NoNewDataState : do / Skip alert triggering process
 NoNewDataState : exit / Return system to idle state


 NoNewDataState --> IdleState : awaitNextSchedule


 CheckingWatchlists : entry / Query all active watchlists
 CheckingWatchlists : do / Match new transactions against subscribed zones
 CheckingWatchlists : exit / Pass matched zones to alert dispatcher


 CheckingWatchlists -right-> SendingAlerts : [zones matched]
 CheckingWatchlists --> NoWatchlistMatchState : [no matches]


 NoWatchlistMatchState : entry / Log that no watchlist zones were affected
 NoWatchlistMatchState : do / Complete update run without sending alerts
 NoWatchlistMatchState : exit / Return system to idle state


 NoWatchlistMatchState --> IdleState : awaitNextSchedule


 SendingAlerts : entry / Compose alert messages for affected users
 SendingAlerts : do / Dispatch email notifications via configured channels
 SendingAlerts : exit / Log alert delivery and return to idle


 SendingAlerts --> IdleState : complete
}
